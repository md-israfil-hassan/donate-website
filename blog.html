Important FAQ

What is the Document Object Model (DOM)?

programming interface for web documents

How do you select an element form the DOM?

getElementById(id)
getElementsByClassName(Classname)
querySelector(cssSelector)

What is event delegation in the context of the DOM and why is it useful?

the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.

How do you mainipulate an element,s attributes and styles using the DOM?

first need to select the desired element using methods like getElementById, querySelector, or similar, then access its properties to modify attribute values with setAttribute and change
